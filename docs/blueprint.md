# **App Name**: MAFA Landing Express

## Core Features:

- Landing Page Generation: Design and develop a static landing page, optimized for conversion and fully responsive across devices.
- Domain & Hosting: Register and configure a domain (.cl, .com) and provide web hosting services for 1 year.
- Contact Integration: Configure direct WhatsApp buttons and links to 3 social media platforms (Instagram, Facebook, TikTok).
- SEO Configuration: Install and configure Title, Meta Description, and Favicon to ensure proper Google indexing.
- Content Update Support: Provide support for minor content updates (text changes, price updates, photo updates) and error correction for the first 12 months.
- Price display: Clearly displays the discounted Black Friday price of $89,990 CLP alongside the original price of $210,000 CLP and a prominent '57% OFF' badge.
- Call to Action Button: Includes a prominent 'Quiero mi Landing Page Ahora' call-to-action button to drive immediate conversions.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey trust and reliability.
- Background color: Light gray (#F5F5F5), for a clean and professional look.
- Accent color: A contrasting orange (#FF9933) for call-to-action buttons and highlights to grab attention.
- Font: 'PT Sans', a humanist sans-serif to create a modern and friendly user experience. It will be used for headlines and body text.
- The layout will follow a single-page structure, with clear sections for the offer, components, pricing, and call-to-action.
- Prioritize clear visual hierarchy with larger headlines and strategic use of white space to ensure readability.
- Use simple, modern icons to represent key features and benefits, enhancing visual communication.
